# Stamp Reuse MVP 产品需求文档

## 1. 产品定义

### 产品名称
Stamp Reuse MVP

### 一句话定位
把现实中的章，快速变成可复用的透明底数字章，并秒盖到任意文件上。

### 英文定位
Turn a real-world stamp into a reusable transparent digital stamp, then place it on any file in seconds.

---

## 2. 为什么现在这样做

普通“做章工具”已经很多，但用户真正愿意为之付费的，不只是“做出一个章”，而是：

1. 把已有的真实印章数字化
2. 变成透明底资产
3. 继续盖到新文件上
4. 直接导出可发送、可提交、可归档的文件

所以本产品不再只是 Seal Generator，而是：

**Document Stamping + Stamp Extraction**

---

## 3. 核心用户

### 3.1 小商家 / 跨境卖家
- 已经在线下或纸质流程里有“自己的章”
- 需要把章快速盖到 invoice、packing list、售后单据上

### 3.2 轻办公用户
- 财务、行政、助理、运营
- 需要在 PDF 或图片文件上放已批准 / 已收款 / 已接收等标识章

### 3.3 创作者 / 工作室
- 有品牌章、合集章、签名章
- 需要在图像与文件中复用

---

## 4. MVP 核心价值

### 模块 A：文件盖章
- 上传 PDF / 图片
- 拖拽章到任意位置
- 调整大小、透明度、旋转角度
- 导出新文件

### 模块 B：旧章提取
- 上传手机拍摄的盖章文件照片
- 自动识别章主体颜色（红 / 蓝）
- 自动去纸张底色，输出透明底 PNG
- 保存为“我的章”并继续复用

### 模块 C：新章生成
- 从模板快速生成可用数字章
- 作为提取能力之外的补充入口

---

## 5. MVP 必须交付的用户流程

### 流程 1：生成新章并盖章
1. 进入 workspace
2. 选择模板并编辑文字
3. 保存到章库
4. 上传一份 PDF 或图片
5. 把章拖到目标位置
6. 导出新文件

### 流程 2：提取旧章并复用
1. 上传手机拍照文件
2. 选择红章 / 蓝章 / 自动识别
3. 调整阈值与清理强度
4. 获得透明底结果
5. 保存为章资产
6. 上传新文件
7. 拖拽章到新文件并导出

---

## 6. 正确的产品承诺

不能承诺：
- 任意手机照片都 100% 完美一键抠章

应承诺：
- 大多数清晰照片可自动提取
- 复杂情况可快速微调

一句话表达为：

**自动提取 + 手动微调 + 一键复用**

---

## 7. 功能范围

### 7.1 第一阶段必须有
- 新章模板生成
- 旧章提取
- 当前会话章库
- PDF / 图片文件上传
- 拖拽放置章
- 调整大小 / 旋转 / 透明度
- 导出图片 / PDF

### 7.2 第二阶段增强
- 透视校正
- 画笔擦除杂点
- 多章自动检测
- 持久化云章库
- 支付门槛与订单管理
- 团队章库与审批流

---

## 8. 收费建议

- 单文件盖章导出：$2.9
- 单次旧章提取并保存：$4.9
- 提取 + 盖章打包：$6.9

MVP 不强调重订阅，先验证单次工作流收费。

---

## 9. 技术方案

### 前端
- Next.js App Router
- Tailwind CSS
- react-pdf + pdfjs-dist

### 后端
- Next.js Route Handlers
- Sharp（图片提取、图片合成）
- pdf-lib（PDF 盖章写回）
- Prisma + PostgreSQL（资产 / 订单 / 项目）

---

## 10. 成功指标

第一轮验证只看三个指标：

1. 有多少用户成功提取出第一枚透明底章
2. 有多少用户完成一次“把章盖到文件上”的动作
3. 有多少用户愿意为导出结果付费

---

## 11. 最终判断标准

如果用户对“旧章提取 + 复用盖章”持续使用，说明产品不是普通工具，而是真正的办公效率产品。
